# CRM system for b2b/b2o.

### Synopsis:
This system was developed for Oyster Telecom, a telecom operator in St. Petersburg.

---

### Features:
- User authorization via SMS
- Kanban boards, cards, checklists, messages, mentions, attachments, tasks, scheduling meetings and calls.
- Warehouse, equipment accounting system, budgeting.
- Group and private notifications
- Integration with 3CX PBX
